# Fantasy_Crowdsale
A simple crowdsale architecture with upgradable token

# installation

```
npm install
```

# run tests

```
npm test
```

or run individual tests:

```
npm test test/crowdsale.js
```

# code coverage

```
npm run coverage
```

# Crowdsale flow

Complete crowdsale flow is as per the describe in
```
test/crowdsale.js
```
